# 籔脇 健司（やぶわき けんじ）

## 概要

**籔脇 健司（やぶわき けんじ）** は、東北福祉大学 健康科学部リハビリテーション学科 **教授**。作業療法士・博士（保健科学）。**包括的環境要因調査票（CEQ）の開発者**として知られ、高齢者の健康関連QOL・環境因子・軽度認知障害（MCI）の予防に関する研究を国際的に展開する第一人者。

X（旧Twitter）では **@yabutfu**（籔脇健司@認定作業療法士／学術誌作業療法副編集長／認知症予防専門士指導者／CEQ開発者）として活発に研究・教育情報を発信中。

## 研究分野・専門領域

### 主要テーマ

高齢者の健康に「作業（occupation）」と「環境」の観点からアプローチする研究を専門とする。具体的には：

- **包括的環境支援**：在宅高齢者が満足した生活を送るための環境評価ツール「CEQ（包括的環境要因調査票）」の開発・活用
- **軽度認知障害（MCI）の予防・リバート研究**：軽度認知障害から正常状態への回復（リバート）を促進する作業療法アプローチの研究
- **高齢者の健康関連QOL**：作業参加・環境因子・身体活動量が高齢者の健康関連QOLに及ぼす影響（構造方程式モデリング等を活用）
- **作業療法教育研究**：職業的アイデンティティ、シングルケースデザイン、OT教育の効果測定

### 研究キーワード

軽度認知障害 / 環境因子 / 健康関連QOL / 高齢者 / 作業療法

### 研究分野（科研費分類）

リハビリテーション科学 / 医療管理学・医療系社会学 / 衛生学・公衆衛生学

## CEQ（包括的環境要因調査票）について

籔脇先生が開発した独自の評価ツール。在宅高齢者の生活環境（物的・社会的・制度的環境など）を包括的に評価し、健康関連QOLの改善に必要な環境支援の指針を提供する。複数の国際論文でその効果が検証されており、中国語版（CEQ-C）の信頼性・妥当性検証など国際展開も進んでいる。

- 研究室サイト: https://ceq.jp/
